# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------- |
| 2.0.x   | ✅                  |
| 1.x     | ⚠️ Security fixes   |

## Reporting Security Vulnerabilities

If you find a security vulnerability, please report it responsibly:

1. **DO NOT** create a public GitHub issue
2. Email: earacicotjr@gmail.com
3. Include detailed reproduction steps

## What We Scan For

KeyFinder detects leaked API keys, tokens, and secrets from:
- Cloud providers (AWS, GCP, Azure)
- Payment processors (Stripe, PayPal)
- Communication platforms (Slack, Discord, Telegram)
- AI services (OpenAI, Anthropic)
- And 80+ other providers

## Disclaimer

This tool is for **authorized security testing only**. Do not use it on websites you don't own or don't have permission to test.
